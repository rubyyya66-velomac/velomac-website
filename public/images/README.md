# Velomac Website Image Guide

Place final photography in these folders:

- `brand/`: official logo assets from the Velomac brochure.
- `hero/`: homepage hero product image or refined product group.
- `products/`: clean product photos on white or light gray background.
- `applications/`: original or approved application visuals.
- `applications/product-use/`: application context visuals used inside product detail pages.
- `factory/`: one factory image and one calibration-related image with no certificate wall treatment.
- `company/`: legacy temporary image folder kept for source continuity.

Recommended product image size: 1200 x 850 px or larger, with the product centered and no text on the image.

Current product images are approved JPEG files stored under `products/`. Original file formats are preserved. Some closest-match files contain embedded Chinese watermarks.

Current application images are approved PNG files stored under `applications/`. Original file formats are preserved.

Product detail application visuals are also preserved as PNG files and stored under `applications/product-use/`.

Footer logo uses `brand/velomac-logo-transparent.png`.
